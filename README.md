# FOP-PROJECT

🎮 Memory Flash Game
A terminal-based memory game written in C with colored ANSI output.

📖 Description
Test your memory! Each round shows ONE new color. You must remember and enter the COMPLETE sequence from Round 1.

🎨 Features
Colored terminal output (RED, GREEN, BLUE, YELLOW)
Random color sequences
🎯 Game Rules
Watch the new color flash on screen
Enter the FULL sequence from Round 1
Each round adds ONE more color
One mistake = Game Over!
Color Codes:

1 = 🟥 RED
2 = 🟩 GREEN
3 = 🟦 BLUE
4 = 🟨 YELLOW
🛠️ Requirements
GCC compiler
Linux/Unix terminal (for ANSI colors)
C standard library
